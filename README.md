# AOPS applicatoin challenge

This is a project put together for the Art of Problem Solving application challenge.

## Getting Started

#### Install dependencies:
```
$ npm install
```

#### Run the build once
```
$ npm run build
```

Or 

#### Run a watcher that you leave open
```
$ npm run watch
```

Start the server
```
$ npm run start
```
